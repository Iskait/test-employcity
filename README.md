## Установка
- Клонировать репозиторий.
- Установить зависимости командой `pnpm install`.
- Запустить сервер командой `pnpm dev`.

## Доступные команды

#### Запуск сервера разработки
```
pnpm run dev
```
#### Билд
```
pnpm run build
```
#### Просмотр билда
```
pnpm run preview
```
#### Линт
```
pnpm run lint
```
### Prettier
```
pnpm run prettier
```
